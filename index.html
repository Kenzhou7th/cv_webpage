Portfolio

USTP Tracker: An IoT-Based University Equipment and Devices Tracker

USTP Tracker is a smart asset-tracking system designed to monitor and manage university-owned equipment such as laptops, PCs, and projectors. By integrating IoT technology, the system provides real-time location updates, usage monitoring, and security alerts to prevent unauthorized movement or loss of devices.

Key Features

  • Real-time Tracking – Uses IoT sensors to track the location and status of university equipment.
  • Usage Monitoring – Logs usage history to ensure proper asset utilization.
  • Security Alerts – Sends notifications for unauthorized movement or potential loss.
  • Admin Dashboard – Provides university staff with an interface for monitoring and managing assets efficiently.

Technology Stack

  • Hardware: IoT Sensors (e.g., RFID, GPS, or BLE)
  • Software: React Native (Mobile App), Node.js (Backend), Firebase/MySQL (Database)
